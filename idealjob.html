Full Stack Developer

I have set myself a goal to become a full stack developer as it feels to be the next step up (or perhaps, a few steps up!) from my web design role. From my understanding, a full stack developer fulfills the role of both a front and back end developer. They are responsible for programming and designing the user facing side of web-based programs whilst also engineering the infrastructure to run them. They are vital for building and maintaining not just websites but also databases, servers and even mobile applications.

It has always been important for me to work for organisations that align with my personal values. As such, I use job boards such as Ethical Jobs to find roles that appeal to my values and skillset. Using this job board I was able to find an advertisement for a full stack developer that I can use as a guide to target my learning and help me achieve my career goal.

Full stack developers are often required to work alongside designers, software engineers and other IT professionals in order to get the job done. As such, teamwork and communication are essential skills to being successful in this role. Fortunately, through my career, I have had many chances to work as part of, and even lead teams. Additionally, I have numerous skills which would be advantageous for this role, including:

  • Web design skills: I have used content management systems and e-commerce platforms such as WordPress and Shopify to build beautiful, functional websites. Through these platforms I have also gained some experience coding with CSS and HTML
  • User experience optimisation skills: I am able to analyse and improve user experience on websites to drive audience engagement and generate business activity
  • Project management skills: I have led teams to successfully deliver numerous projects that met budgets and deadlines whilst exceeding client expectations

In order to become a full stack developer, I will need to expand my front-end development skills and master back-end development. Furthermore, like any job, experience in a similar role is often a prerequisite. I will endeavour to network with industry professionals and seek internships and other learning opportunities to practice my skills in a real-world context. I will increase my chances of landing these opportunities by applying myself to my studies and continually developing my skills in personal projects.

Understanding and excelling in front and back end development will be an enduring task. In order to achieve my goals, I established the below plan to break the subject matter down into digestible chunks and the tools I can use to advance my learning in each area.

Learning Plan

Topics

  • Programming languages (visual and back-end)
  • Testing and debugging
  • User interface/user experience design
  • Frameworks (JavaScript and back-end)
  • Database systems and SQL
  • Cyber Security
  • APIs

Methods of Learning

  • Attend university classes to learn the elements of programming and seek advice from experts
  • Analyse the user interface and experience of a wide range of programs to understand the elements of good (and poor) design
  • Build mock web programs through GitHub to develop my coding skills and to practice testing and debugging code
  • Explore frameworks such as Django or Rails and utilise them to begin practicing building web applications with different programming languages
  • Study SQL Server and related programs to practice coding database systems to integrate into my programs
  • Explore jQuery to study existing JavaScript programs and practice customising the code to create something new
  • Build mock web programs that harness APIs to communicate with existing programs to extract data and increase the functionalities of my program
  • Consider and research different security software systems to understand their role in cyber security and how to integrate them into my programs
  • When I encounter problems that I cannot solve, participate in online forums (through Canvas, reddit etc) to troubleshoot and learn from my peers
